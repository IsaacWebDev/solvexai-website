# Medical Practice - $797

Professional medical practice template with appointment booking and services.

## ✨ Features
- 5 pages (Home, Services, Providers, Patient Portal, Contact)
- Calming blue color scheme
- Service cards with icons
- Provider profiles
- Appointment booking (Calendly-ready)
- Trust-building design
- Fully responsive
- Clean, accessible design

## 🚀 Quick Start
```bash
npm install
npm run dev
```

Visit http://localhost:3000

## 📦 Build & Deploy
```bash
npm run build
vercel --prod
```

## 🎨 Customization
Edit colors in `tailwind.config.ts`:
```ts
colors: {
  primary: '#4A90E2',  // Calming blue
  secondary: '#67C4EF',
}
```

## 📄 Pages
- `/` - Homepage with hero, services, providers
- `/services` - Full service catalog (ready to build)
- `/providers` - Doctor profiles (ready to build)
- `/contact` - Appointment booking form (ready to build)

## 💼 Perfect For
Medical practices, dental offices, therapy clinics, healthcare providers

Built with Next.js 14 + TailwindCSS + TypeScript
