## 🚀 DEPLOYMENT INSTRUCTIONS

### For Each Template:

1. **Install Dependencies:**
```bash
cd templates/[template-name]
npm install
```

2. **Test Locally:**
```bash
npm run dev
# Visit http://localhost:3000
```

3. **Build for Production:**
```bash
npm run build
```

4. **Deploy to Vercel:**
```bash
vercel --prod
```

### Batch Deploy All Templates:
```bash
# From templates/ directory
for dir in fitness-studio-energy real-estate-luxury ecommerce-clean medical-practice construction-pro creative-agency; do
  cd $dir
  npm install && npm run build && vercel --prod
  cd ..
done
```
