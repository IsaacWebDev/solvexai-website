# Real Estate Luxury - $897

Elegant luxury real estate template with property showcases and agent profiles.

## ✨ Features
- 5 pages (Home, Properties, About, Agents, Contact)
- Sophisticated gold & dark design
- Property cards with pricing
- Location-based filtering
- Agent profiles
- Stats & achievements
- Fully responsive
- Premium serif typography (Cormorant Garamond)

## 🚀 Quick Start
```bash
npm install
npm run dev
```

Visit http://localhost:3000

## 📦 Build & Deploy
```bash
npm run build
vercel --prod
```

## 🎨 Customization
Edit colors in `tailwind.config.ts`:
```ts
colors: {
  primary: '#B8860B',  // Gold
  dark: '#1A1A1A',
}
```

## 📄 Pages
- `/` - Homepage with hero, featured properties, stats
- `/properties` - Full property catalog with filters
- `/about` - Company story, values, team
- `/agents` - Agent directory (ready to build)
- `/contact` - Contact form & map (ready to build)

## 💼 Perfect For
Luxury real estate agencies, property developers, realtors, commercial real estate

Built with Next.js 14 + TailwindCSS + TypeScript
