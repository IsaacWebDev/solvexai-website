# SolveXAI Website - 10/10 Rebuild Plan

## Isaac's Vision (Mandatory Requirements)

### Homepage (Apple-style Minimal)
- **ONLY**: DNA helix video background + SolveXAI logo centered
- **NO**: services, text clutter, CTAs, or any other content
- Minimal nav bar (appears on scroll only)
- Immersive, cinematic, minimal

### Separate Pages
- `/services` - Templates, custom dev, AI receptionist
- `/about` - Team, mission, stats
- `/contact` - Inquiry form, booking

### Technical Requirements
- Full SEO (meta tags, structured data, sitemap, robots.txt)
- Performance: 90+ Lighthouse, <2s load, 60 FPS
- Mobile responsive
- Next.js 14+ App Router

## Execution Strategy

### Phase 1: Core Architecture (senior-dev)
- Setup Next.js routing structure
- Configure SEO foundation (layout.tsx, metadata)
- Create sitemap.ts and robots.ts
- Setup performance optimizations

### Phase 2: Homepage (frontend + ui-designer + vision)
- **vision**: Generate DNA helix video with Kling OR create fallback
- **ui-designer**: Generate SolveXAI logo with Nano Banana Pro
- **frontend**: Implement minimal homepage with video background + logo
- **frontend**: Build scroll-triggered navigation component

### Phase 3: Content Pages (frontend)
- `/services` page with service cards
- `/about` page with stats
- `/contact` page with form + API route

### Phase 4: Optimization (perf-bench)
- Video compression (<5MB)
- Image optimization (WebP/AVIF)
- Font optimization (next/font)
- Lighthouse audit

### Phase 5: Final Verification (reality-checker)
- Verify all 10/10 requirements met
- Check homepage has ONLY DNA helix + logo
- Verify separate pages exist
- Test navigation behavior
- Performance benchmarks
- Mobile responsiveness
